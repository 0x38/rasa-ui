1. API
RASA API

We offer a "pass through" API for the Rasa NLU API, the reason Rasa UI does this, is to provide logging of the requests and types, for future and further analsysis to improve your training.

Your Application <--> Rasa UI Middleware <--> Rasa NLU

Examples

POST /parse

{{parse_cmd}} 
curl -XPOST localhost:5000/parse -d '{"q":"hello there"}'

POST /train
